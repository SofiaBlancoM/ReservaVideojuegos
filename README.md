# Aplicación Reserva de Videojuegos

## Descripción
Aplicación de escritorio desarrollada en JavaFX para la gestión y reserva de videojuegos.  
Permite a los usuarios consultar videojuegos disponibles y realizar reservas de forma sencilla.

## Tecnologías implicadas
- Java 21
- JavaFX
- Maven
